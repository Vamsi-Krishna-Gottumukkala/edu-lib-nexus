# Edu-Lib Nexus

Edu-Lib Nexus is a modern Library Management System built for educational institutions.

## Features
- **RFID Integration:** Fast and seamless student and faculty library access tracking.
- **Library Management:** Easily track, issue, and manage library books.
- **Attendance Logs:** Track daily attendance automatically.
- **Question Paper Repository:** Securely store and share academic resources.
- **Role-Based Access Control:** Distinct views and features for Super Admins, Branch Admins, Students, and Faculty.

## Tech Stack
- Frontend: React, Vite, TailwindCSS, shadcn/ui
- Backend: Node.js, Express, Supabase (PostgreSQL)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Setup `.env` variables for Supabase and Backend API.
4. Run frontend: `npm run dev`
5. Run backend: `cd backend && node server.js`
